Identglue example

The library can detect if you are on mobile or desktop:

Platform:

Browser info:

Desktop

A request is being performed every few seconds to check if you have the Ausweiss App installed and opened.

Status: loading

The status can have one of the following values:
  • unavailable: The app is not opened
  • available: The app is opened
  • safari: If you are on safari browser and the library cannot detect if the app is opened or not

Mobile

On mobile platform we cannot detect if you have the app installed or not. We are checking to see if the app opens when you click the link below. If you don't have the app installed then we will show you the download links.
